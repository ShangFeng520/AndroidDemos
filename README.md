## 三：MVVM架构 SunnyWeather

**项目描述：** 一款天气查询app，可以搜索全球各个城市的天气信息。整体采用MVVM架构，使用了ViewBinding、LiveData、ViewModel、Retrofit2、Gson、kotlin coroutine、sharedPreferences、下拉刷新、滑动菜单等技术。
